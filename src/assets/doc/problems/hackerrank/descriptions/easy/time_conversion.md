# Time Conversion

## Description

Given a time in `12`-hour AM/PM format, the problem involves converting it to military (`24`-hour) time. The goal is to return a new string representing the input time in a `24`-hour format.

## Function

```java
public static String timeConversion(String s) {}
```

## Input Format

- A single string representing a time in `12`-hour clock format (e.g., `hh:mm:ssAM` or `hh:mm:ssPM`).

## Output Format

- A string representing the time in `24`-hour format.

## Constraints

- All input times are valid.

## Example

```bash
# Sample Input
07:05:45PM

# Sample Output
19:05:45
```
